Lei Zhang Portfolio Website
Description

This is a professional portfolio website for Lei Zhang, a Full-Stack Developer and MS Information Systems student. The site showcases personal profile information, skills, education, projects, testimonials, and a contact form. The website is fully responsive for desktops, tablets, and mobile devices.

HTML Tags Used and Descriptions

<!DOCTYPE html> : Declares the HTML5 document type.

<html> : Root element of the HTML document.

<head> : Contains metadata, links to CSS, title, and favicon.

<meta> : Provides metadata like charset and viewport settings.

<title> : Sets the page title displayed in the browser tab.

<link> : Links external resources (CSS, favicon).

<body> : Contains the visible content of the page.

<header> : Defines the header section, usually contains navigation.

<h1> - <h2> - <h3> : Headings of various levels to structure content.

<nav> : Navigation links.

<a> : Hyperlinks to other pages or external sites.

<div> : Generic container for layout purposes (used for .main-content, .gallery, .testimonial-grid).

<section> : Groups related content within the main area (About, Projects, Testimonials, Contact).

<article> : Represents self-contained content (Education table).

<aside> : Sidebar content (Skills).

<p> : Paragraphs of text.

<ul> : Unordered list.

<li> : List items inside <ul>.

<form> : Form for user input (Contact section).

<label> : Labels for form input fields.

<input> : Input fields (text, email).

<textarea> : Multiline text input for messages.

<button> : Clickable submit button.

<table> : Displays tabular data (Education).

<tr> : Table row.

<th> : Table header cell.

<td> : Table data cell.

<figure> : Container for images and captions (Projects, Testimonials).

<img> : Displays images.

<figcaption> : Caption for the <figure> image.

<footer> : Footer section with contact info and copyright.

Features:

Sticky navigation bar that remains visible while scrolling.

Responsive layout using Flexbox and CSS Grid.

Skills sidebar displayed next to profile on desktop, stacked on mobile.

Projects gallery with hover effects and clickable links.

Testimonials section with hover effects.

Education table styled with CSS selectors.

Contact form fully responsive on tablets and mobile devices.
