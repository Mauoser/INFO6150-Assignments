Anime Pilgrimage Website

Description:
Anime Pilgrimage is a website that connects anime fans with real-life locations in Japan that inspired iconic scenes from popular anime. The site includes information about locations, travel guides, community sign-up, and multimedia elements like images, audio, and videos.

HTML Tags Used and Descriptions
<!DOCTYPE html> : Declares the HTML5 document type.
<html> : Root element of the HTML document.
<head> : Contains metadata, links to CSS, title, and favicon.

<meta> : Provides metadata like charset and viewport settings.

<title> : Sets the page title displayed in the browser tab.
<link> : Links external resources (CSS, favicon).
<body> : Contains the visible content of the page.
<header> : Defines the header section, usually contains navigation.
<h1> - <h3> : Headings of various levels to structure content.
<nav> : Navigation links.

<a> : Hyperlinks to other pages or external sites.

<main> : Main content area of the page.
<section> : Groups related content within the main area.
<article> : Represents a self-contained content block (e.g., a location).
<aside> : Sidebar content or extra information.
<p> : Paragraphs of text.
<ul> : Unordered list.
<li> : List items inside <ul> or <ol>.
<form> : Form for user input (signup).

<label> : Labels for form input fields.

<input> : Input fields (text, email, password, etc.).

<datalist> : Provides predefined options for input.

<option> : Options inside <datalist> or <select>.

<button> : Clickable button.

<table> : Displays tabular data.
<tr> : Table row.
<th> : Table header cell.
<td> : Table data cell.
<figure> : Container for images and captions.

<img> : Displays images.

<figcaption> : Caption for the <figure> image.
<details> : Creates an expandable/collapsible content section.
<summary> : Summary or title for <details>.

<video> : Embeds a video player.

<source> : Specifies video or audio source.

<audio> : Embeds an audio player.

<footer> : Footer section with contact info and copyright.
